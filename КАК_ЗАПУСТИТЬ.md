# 🚀 Инструкция по запуску сайта на Vercel

## Что тебе понадобится
- Аккаунт на **GitHub** (бесплатно) → github.com
- Аккаунт на **Vercel** (бесплатно) → vercel.com

---

## Шаг 1 — Загрузи проект на GitHub

1. Зайди на **github.com** → нажми **"New repository"** (зелёная кнопка)
2. Назови репозиторий, например: `my-consulting-site`
3. Выбери **Private** (чтобы код не был публичным)
4. Нажми **"Create repository"**
5. На следующей странице нажми **"uploading an existing file"**
6. Перетащи **все файлы из этой папки** в окно загрузки
   - ⚠️ Важно: загружай папку `src` целиком и все остальные файлы
7. Нажми **"Commit changes"**

---

## Шаг 2 — Подключи Vercel

1. Зайди на **vercel.com** → нажми **"Sign Up"** → выбери **"Continue with GitHub"**
2. После входа нажми **"Add New Project"**
3. Найди свой репозиторий `my-consulting-site` → нажми **"Import"**
4. Настройки оставь по умолчанию — Vercel сам определит что это Vite/React
5. Нажми **"Deploy"**
6. Через 1-2 минуты сайт готов! Ты получишь ссылку вида:
   `https://my-consulting-site.vercel.app`

---

## Шаг 3 — Подключи свой домен (опционально)

1. В Vercel перейди в **Settings → Domains**
2. Введи свой домен, например `petrov-coo.ru`
3. Vercel покажет DNS-записи — добавь их в настройках у регистратора домена
4. Через 10-30 минут сайт будет доступен по твоему домену

---

## Как обновлять сайт

### Вариант А — Через редактирование кода
1. Открой файл `src/App.jsx`
2. Найди объект `C0` в начале файла — там всё содержимое сайта
3. Измени нужный текст, сохрани файл
4. Загрузи обновлённый `App.jsx` на GitHub (в тот же репозиторий)
5. Vercel автоматически пересоберёт сайт за 1-2 минуты

### Вариант Б — Через режим редактирования на сайте
- Нажми кнопку **"✏️ Редактировать"** на сайте
- Кликай на любой текст и меняй прямо в браузере
- ⚠️ Эти изменения сохраняются только в текущей сессии браузера
- Чтобы сохранить навсегда — скопируй изменённые тексты в `App.jsx` и загрузи на GitHub

## Как добавить своё фото

1. Загрузи фото на любой хостинг:
   - **imgbb.com** (бесплатно, просто)
   - **Google Drive** → правой кнопкой → "Открыть доступ" → "Все у кого есть ссылка"
2. Скопируй прямую ссылку на изображение
3. В файле `src/App.jsx` найди блок с `👤` и `"Ваше фото"`
4. Замени заглушку на тег: `<img src="ССЫЛКА_НА_ФОТО" style={{width:"100%",height:"100%",objectFit:"cover"}} />`
